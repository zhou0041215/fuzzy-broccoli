Put optional bundled resume fonts here as WOFF2 files.

Supported file names:

- NotoSansSC-Regular.woff2
- NotoSansSC-Bold.woff2
- NotoSerifSC-Regular.woff2
- NotoSerifSC-Bold.woff2
- HarmonyOSSansSC-Regular.woff2
- HarmonyOSSansSC-Bold.woff2
- LXGWWenKai-Regular.woff2
- LXGWWenKai-Bold.woff2

If a file is missing, FlowCV falls back to system fonts automatically.
